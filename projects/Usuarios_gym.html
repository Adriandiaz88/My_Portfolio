Descripción

El objetivo de este proyecto es mejorar la retención de usuarios mediante análisis de comportamiento y clustering, identificando patrones clave y desarrollando estrategias personalizadas.

Metodología

Preprocesamiento

  • Manejo de valores nulos
  • Feature engineering

Análisis Exploratorio

  • DAU/MAU
  • Frecuencia de visitas
  • Duración de sesiones

Modelado

  • Clustering jerárquico
  • Segmentación RFM
  • Análisis de cohortes

Machine Learning

Clustering, SciPy
